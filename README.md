# adsvitorcosta - IA aplicada

Central editorial sobre inteligencia artificial aplicada a e-commerce, dropshipping e agencias.

## Estrutura

- Home organizada por topicos.
- Paginas proprias para Modelos, Ferramentas, E-commerce, Dropshipping, Automacoes e Tutoriais.
- Artigos visuais com fontes, comparacoes, graficos e testes praticos.

## Rodar localmente

```bash
npm install
npm run dev
```

## Validar

```bash
npm run build
```
